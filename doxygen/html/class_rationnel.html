Constante * Rationnel::cosinus ( bool  angle) [virtual]

Cosinus Rationnel

La fonction cos de cmath étant implémentée pour un float, on fait le rapport du numérateur sur le dénominateur pour commencer. Ensuite, on teste si on est en degré. Si c'est le cas, on multiplie le résultat par PI/180 car cos est définie en radians. On calcule ensuite le cosinus.

Paramètres:
angleBooléen, permet de savoir si l'on est en mode degré
Renvoie:
Un nouveau Complexe construit à partir du Rationnel contenant le résultat
